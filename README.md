# 项目结构
* [传统项目结构]()
基于dao、service、controller的三层架构。是非常容易理解的，也是最基础的架构。形如一条链，不断的
向下调用。在一般业务中是可以这样的，但是如果业务中存在许多的嵌套调用，就会非常麻烦，且如果业务需要
变化在dao修改了代码，与之有关的service层代码几乎都需要重写，十分的不便。
